Systemlessly replaces emoji font with different emojis

Based on [Keinta15/Magisk-iOS-Emoji](https://github.com/Keinta15/Magisk-iOS-Emoji)

## Build

To simply build the files just use one of the following commands:
```sh
./buildNoto.sh -b
./buildIOS.sh -b
./buildTwemoji.sh -b
./buildBlobmoji.sh -b
```

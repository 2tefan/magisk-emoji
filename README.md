# Magisk Emoji
Systemlessly replaces emoji font with different emoji

Based on [Keinta15/Magisk-iOS-Emoji](https://github.com/Keinta15/Magisk-iOS-Emoji)

## Build

```sh
./buildNoto.sh
```
or
```sh
./buildIOS.sh
```
